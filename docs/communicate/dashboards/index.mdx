# Dashboards Overview

Dashboards provide a comprehensive view of your application's health and performance. Checkly's dashboard functionality allows you to create custom visualizations and share insights with your team and stakeholders.

## What are Dashboards?

Dashboards are customizable views that display key metrics, charts, and status information in one place. They help you:

- Monitor application health at a glance
- Share insights with team members and stakeholders
- Track performance trends over time
- Identify patterns and anomalies quickly

## Key Features

### Customizable Widgets
- **Metrics Widgets**: Display key performance indicators
- **Status Widgets**: Show current service status
- **Chart Widgets**: Visualize data trends and patterns
- **Custom Widgets**: Add external data and integrations

### Flexible Layout
- **Drag-and-drop**: Easy widget arrangement
- **Responsive Design**: Optimized for different screen sizes
- **Custom Styling**: Match your brand and preferences
- **Real-time Updates**: Live data refresh

### Sharing Optionss
- **Team Access**: Control who can view and edit
- **Public Sharing**: Share with stakeholders
- **Embedding**: Integrate into other applications
- **Export Options**: Generate reports and exports

## Getting Started

To create a dashboard:

1. Navigate to the Dashboards section
2. Choose a template or start from scratch
3. Add and configure widgets
4. Arrange the layout to your preference
5. Set up sharing and access permissions

## Dashboard Types

### Operational Dashboards
Focus on real-time operational metrics and current system status.

### Executive Dashboards
Provide high-level overview for management and stakeholders.

### Technical Dashboards
Show detailed technical metrics for engineering teams.

### Custom Dashboards
Tailored dashboards for specific use cases and teams.
