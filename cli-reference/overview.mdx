## Installing

You can use the Checkly CLI without installing globally by using npx:

```bash
npx checkly --help
```

Or install it globally with npm:

```bash
npm install -g checkly
```

## Updating

If installed globally, update the CLI with:

```bash
npm update -g checkly
```

If using npx, you always get the latest version.

## Version

To check your installed Checkly CLI version:

```bash
npx checkly --version
```
